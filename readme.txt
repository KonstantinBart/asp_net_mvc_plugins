Используется ASP.NET MVC 5, Autofac, jQuery Ajax, JS, Bootstrap.

Запускаем приложение (AspNetMvcPlugins) и выбираем:
- папку для поиска,
- необходимость учитывать поиск в подкаталогах,
- максимальный размер файлов (в Кб),
- дату создания файлов (до),
- атрибуты файлов.

В выпадающем списке выбираем модуль и под ним отображается представление для выбранного модуля.
Все доступные модули находятся в папке Plugins, а модули для отображения в выпадающем списке - в папке SelectedPlugins (в эту папку мы можем добавлять/удалять требуемые модули и в течение 10 сек максимум выпадающий список обновится).

При нажатии на кнопку поиска происходит выборка списка файлов, подходящих по условию, и динамическое наполнение списка результатов поиска.
